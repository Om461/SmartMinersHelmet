Project Description: Smart Mining Helmet Dashboard

The Smart Mining Helmet Dashboard is a sophisticated, real-time web application designed to monitor the safety and environmental conditions of mining personnel simultaneously. Built utilizing modern web technologies—specifically React.js and Vite—this application serves as the centralized command center for mining managers to ensure the well-being of their workforce in hazardous environments.



At its core, the dashboard processes and visualizes complex telemetry data simulated from internet-of-things (IoT) enabled "Smart Helmets." It tracks critical vital signs, including heart rate and body temperature, alongside environmental hazards such as ambient temperature, humidity, and explosive gas levels (LEL).



The user interface was crafted from the ground up using a custom CSS variable-based design system, emphatically avoiding pre-built UI libraries to demonstrate raw front-end proficiency. The aesthetic features a premium, dark-industrial "cyber" theme utilizing glassmorphism techniques, smooth micro-animations, and dynamic neon accents that immediately convey health statuses (Safe, Warning, Critical) at a glance.



Key features include:
Real-Time Status Grid: A responsive overview of all active miners, instantly filterable by current safety status or searchable by name and ID.

Deep-Dive Analytics: Interactive modal views for individual miners that present detailed metrics through visually intuitive progress bars and indicators.

Emergency Alert System: A robust, floating notification engine that triggers immediate warnings when crucial thresholds—such as hazardous gas detection or extreme resting heart rates—are breached.

Custom State Management: A complex React hook (useMiners) engineered to simulate realistic, continuous data streams and evaluate complex safety thresholds on the fly.

This project showcases a strong command of modern React concepts, state management, responsive UI/UX principles, and the ability to design high-fidelity, performant administrative interfaces for mission-critical applications.
